Agentic AI & GenAI

Master Agentic AI and Generative AI by building intelligent systems, NLP models, and conversational agents using frameworks like LangChain and Vertex AI.

Course Plan

  • Week 1: Introduction to Agentic AI
    • Explore the fundamentals of Agentic AI and its role in problem-solving
  • Week 2: Phi Data: Agentic AI Framework
    • Learn Phi Data framework for building Agentic AI systems
  • Week 3: LangChain
    • Understand LangChain for creating complex AI applications with LLMs
  • Week 4: LCEL (LangChain Expression Language)
    • Work with LangChain Expression Language for efficient LLM workflows
  • Week 5: LangServe for Efficient AI Deployment
    • Deploy and scale AI applications using LangServe
  • Week 6: LangGraph
    • Manage complex AI workflows with LangGraph
  • Week 7-8: Text Summarization with AWS Services
    • Create text summarization apps using AWS Lambda and Bedrock
  • Week 9-10: Fine-Tuning Foundation Models on Custom Data
    • Fine-tune foundation models using AWS SageMaker
  • Week 11-12: Project: AWS
    • Build a Retrieval-Augmented Generation (RAG) system and chatbot on AWS
  • Week 13: GCP Basics & Introduction to Vertex AI
    • Explore Google Cloud and Vertex AI for AI model deployment
  • Week 14-15: Gemini Models with Vertex AI and Google AI Studio
    • Leverage Gemini models for advanced AI applications
  • Week 16-17: Project: GCP
    • Implement RAG and build a chatbot using Gemini Pro on GCP
  • Week 18: Capstone Project I
    • Design an Agentic AI system integrating multiple frameworks
  • Week 19: Capstone Project II
    • Develop and deploy a GenAI conversational agent
  • Week 20: Final Review and Deployment
    • Review projects, optimize, and deploy solutions
